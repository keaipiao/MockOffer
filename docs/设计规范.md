## 1. 设计基调

**一句话**：专业可信、年轻有科技感的 AI 求职平台——**苹果液态玻璃质感 + 灵动岛悬浮导航**，克制不浮夸（求职是严肃场景，玻璃与渐变只做点睛，不喧宾夺主）。

- **关键词**：玻璃通透 / 克制科技感 / 大圆角 / 充足留白 / 层级清晰
- **明暗**：**浅色为主**（清爽专业亲和），深色为辅（科技/AI 感，作为可切换主题）
- **标志性元素**：① 半透明磨砂玻璃卡片（glassmorphism）；② 顶部居中、非通栏的**灵动岛式悬浮胶囊导航**；③ 靛蓝→青品牌渐变只用于重点/CTA/光晕
- **品牌 Logo**：扁平渐变「M」字标（取自「模考」）——一条圆润缎带交叠折成的 M，圆头端点、中央缎带交叠穿插、谷深约 70%；配色为靛蓝→青对称渐变（左腿底深蓝、两峰浅、中央青、右腿底浅蓝），纯扁平、无玻璃质感 / 光影、透明背景、无文字。横版锁版 ＝ 该 M ＋「模考Offer」深色粗体（`#0F172A`）。母版见 `docs/资源/logo/png/`（`icon-M.png` 图标母版、`horizontal-light-M.png` 横版母版）；站点资产（favicon / app 图标 / 横版 logo）在 `frontend/public/`。（旧 MO 玻璃图标资产 `icon.png` / `lockup-*` / `horizontal-light.png` / `refined/` / `concept-*` 已全部移除。）
- **参考气质**：Apple 玻璃质感、Linear、Vercel 的现代克制
- **品牌引流位**：页脚 / 关于页固定「本项目由 [灵计一动](https://lingji1dong.com) 驱动开发」徽章位，作为全站统一元素（不单列功能，落到组件与页脚）
- 设计方向由 6 张 AI 参考样板定调（见 `docs/资源/样板/`，§9 路线 C）；样板仅定视觉方向，落地用真实 HTML/CSS 重建。

## 4. 间距 / 圆角 / 阴影

> 仅先锚定与玻璃语言强相关的圆角与玻璃投影；间距刻度先给一套基线，其余边做边长。

- **间距刻度**：4 / 8 / 12 / 16 / 24 / 32 / 48（统一用这套，8 为基准）
- **圆角**：小 `12px`（输入框/小按钮）· 中 `16px` · 大 `24px`（玻璃卡片/弹窗）· 胶囊 `999px`（灵动岛导航/主按钮）
- **阴影**：玻璃卡柔和投影 `0 8px 32px rgba(15,23,42,0.10)`（浅）/ `0 8px 32px rgba(0,0,0,0.40)`（深）；配 1px 内高光边模拟玻璃厚度

## 5. 组件规范

> 以下为 M1 账号体系沉淀的通用组件，后续功能复用、不重复造；功能特有组件在各功能 `设计.md` §3。

### 5.1 玻璃卡 / 面板
半透明白 `rgba(255,255,255,0.65)` + `backdrop-filter: blur(20px)` + 1px 高光内描边 `inset 0 1px 0 rgba(255,255,255,0.92)` + 投影 `0 8px 32px rgba(15,23,42,0.10)`；圆角 24（大卡）/ 16（小面板）。

### 5.2 悬浮胶囊导航（灵动岛）
顶部居中、非通栏、**宽度自适应内容**、固定高度（约 60）、大圆角（约 30）。元素从左到右用 1px 竖线分割：Logo（横版锁版）｜主导航项｜右侧用户头像。导航项 = 图标 + 文字；**当前项选中态 = 白底 + 品牌色文字 + 轻投影**胶囊。头像点击展开下拉（头像 + 昵称 + 邮箱 + 菜单项）。

### 5.3 按钮
- **主按钮**：品牌渐变实心，高 56（表单）/ 38（行内），圆角 12；hover 抬升 1px + 阴影加深。
- **次按钮**：白底 + 1px 描边（边框色），hover 提亮 + 描边加深。
- **品牌描边强调**：`1px 主色描边 + 主色文字 + 主色 8% 底`，用于次级正向操作（如「绑定」），不与主 CTA 抢权重。
- **禁用**：浅灰底 + 弱文字 + `not-allowed`。

### 5.4 输入框
玻璃浅底 + 1px 轻描边（边框色），圆角 12，高 56；`:focus-within` 描边转主色 + 3px 品牌色光环。

### 5.5 6 位验证码
等宽 6 格、圆角 12；已填态品牌色描边 + 浅光环；当前激活格主色描边 + 光环。

### 5.6 设置页骨架
左侧分类侧栏 + 右侧内容，**合并为一张玻璃卡、中间 1px 竖线分隔**。侧栏：分组标题（弱化色）+ 子项缩进 + 选中胶囊（主色 8% 底）；未上线项灰显 + 「即将支持」。内容区列表行：图标盒 + 名称（可带徽章）+ 副文 + 右侧操作，行间分隔线，**不嵌套卡片**。

### 5.7 徽章
胶囊小标签，圆角 999、高 22；品牌色（主色 + 8% 底）用于强调状态，中性灰用于次要 / 未启用状态。

## 6. 响应式

- 内容容器最大宽 1040、居中。
- 认证卡 `min(420px, 100%)`；移动断点 520px（卡片近满宽、留 16–24px 边距、CTA 通栏）。
- 设置页两栏，移动断点 860px 收为单列（侧栏转横向、内容近满宽）。
- 悬浮胶囊导航移动端可隐藏导航项文字、保留图标。
