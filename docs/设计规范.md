## 1. 设计基调

**一句话**：专业可信、年轻有科技感的 AI 求职平台——**苹果液态玻璃质感 + 灵动岛悬浮导航**，克制不浮夸（求职是严肃场景，玻璃与渐变只做点睛，不喧宾夺主）。

- **关键词**：玻璃通透 / 克制科技感 / 大圆角 / 充足留白 / 层级清晰
- **明暗**：**浅色为主**（清爽专业亲和），深色为辅（科技/AI 感，作为可切换主题）
- **标志性元素**：① 半透明磨砂玻璃卡片（glassmorphism）；② 顶部居中、非通栏的**灵动岛式悬浮胶囊导航**；③ 靛蓝→青品牌渐变只用于重点/CTA/光晕
- **品牌 Logo**：MO 玻璃图标（M 模考 + O Offer + 对话气泡 + 对勾，呼应「模拟面试→拿到 Offer」）+「模考 / OFFER」双行锁版；英文用全大写 OFFER 与方块中文等重，避免小写尾巴失衡。母版见 `docs/资源/logo/png/`（`icon.png` 图标、`lockup-light/dark.png` 主锁版、`horizontal-light.png` 横版），定稿渲染在 `refined/`；站点资产（favicon / icon / 横版 logo）在 `frontend/public/`。
- **参考气质**：Apple 玻璃质感、Linear、Vercel 的现代克制
- **品牌引流位**：页脚 / 关于页固定「本项目由 [灵计一动](https://lingji1dong.com) 驱动开发」徽章位，作为全站统一元素（不单列功能，落到组件与页脚）
- 设计方向由 6 张 AI 参考样板定调（见 `docs/资源/样板/`，§9 路线 C）；样板仅定视觉方向，落地用真实 HTML/CSS 重建。

## 3. 字体

- **字族**（中西混排，西文在前优先匹配）：
  - 正文 / 标题：`"Inter", "PingFang SC", "HarmonyOS Sans SC", "Microsoft YaHei", system-ui, sans-serif`
  - 等宽（数字 / 代码 / 分数）：`"JetBrains Mono", ui-monospace, monospace`
- **字号层级**（字号 / 行高 / 字重）：

| 层级 | 字号 | 行高 | 字重 |
|---|---|---|---|
| Display（落地大标题）| 48 | 1.1 | 700 |
| H1 | 32 | 1.2 | 700 |
| H2 | 24 | 1.3 | 600 |
| H3 | 20 | 1.4 | 600 |
| 正文 Body | 16 | 1.6 | 400 |
| 小字 Small | 14 | 1.5 | 400 |
| 辅助 Caption | 12 | 1.5 | 400 |

## 4. 间距 / 圆角 / 阴影

> 仅先锚定与玻璃语言强相关的圆角与玻璃投影；间距刻度先给一套基线，其余边做边长。

- **间距刻度**：4 / 8 / 12 / 16 / 24 / 32 / 48（统一用这套，8 为基准）
- **圆角**：小 `12px`（输入框/小按钮）· 中 `16px` · 大 `24px`（玻璃卡片/弹窗）· 胶囊 `999px`（灵动岛导航/主按钮）
- **阴影**：玻璃卡柔和投影 `0 8px 32px rgba(15,23,42,0.10)`（浅）/ `0 8px 32px rgba(0,0,0,0.40)`（深）；配 1px 内高光边模拟玻璃厚度

## 9. UI 出图规范

> 本项目 M0 定调采用**路线 C · AI 生图作参考**：用 codex 内置 image_gen（gpt-image-2）生成 6 张高保真参考样板（`docs/资源/样板/` 首页/登录/工作台 × 深浅），定下视觉方向。
